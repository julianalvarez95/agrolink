## 🔥 El problema

El campo argentino tiene un problema silencioso: **la gente existe, el trabajo existe, pero nadie los conecta de forma confiable.**

En el noroeste bonaerense, cada campaña (soja, trigo, maíz, girasol) abre una ventana de tiempo crítica. Si no conseguís un maquinista en 72 horas, perdés días de cosecha. Si sos trabajador y no tenés contactos, te quedás afuera de las oportunidades.

```
┌─────────────────────────────────────────────────────────┐
│              📊  Los números del dolor                  │
├──────────────────────┬──────────────────────────────────┤
│  70% de productores  │  dificultad para conseguir       │
│                      │  personal calificado en campaña  │
│                      │  📌 Bolsa de Comercio Rosario    │
├──────────────────────┼──────────────────────────────────┤
│  90% de contratistas │  reporta escasez de maquinistas  │
│                      │  durante cosecha gruesa          │
│                      │  📌 FACMA                        │
├──────────────────────┼──────────────────────────────────┤
│  100% de consultados │  no tiene un canal confiable     │
│  en diagnóstico      │  para contratar o conseguir      │
│  local               │  trabajo rural                   │
│                      │  📌 Diagnóstico AgroLink         │
└──────────────────────┴──────────────────────────────────┘
```

El mercado opera de forma **completamente informal**: boca a boca, mensajes de WhatsApp, favores entre conocidos. No hay verificación de referencias. No hay formalización. No hay escala.

## 🏗️ Nuestra apuesta

AgroLink no es una app. Es un servicio de intermediación construido sobre personas reales que verifican, conectan y acompañan.

**Tres pilares:**

| Pilar | Qué significa |
|-------|---------------|
| ✅ Verificación | Cada trabajador tiene 3 referencias laborales chequeadas. Sabés a quién contratás. |
| ⚡ 48 horas | Nos decís qué necesitás y te presentamos candidatos. Sin vueltas. |
| 📚 Capacitación | Formamos operarios en maquinaria actual, agricultura de precisión y manejo ganadero. |

**¿Por qué low-tech on purpose?**

No queremos construir una plataforma antes de entender el problema. El campo tiene sus propios ritmos, sus propios canales de confianza. Primero escuchamos. Después construimos.

## 📍 Etapa actual: Ideación → Validación

Estamos en la etapa más honesta de cualquier proyecto: **no sabemos si esto funciona todavía. Pero lo estamos averiguando.**

Esta landing no es un producto. Es un experimento. La estamos circulando entre productores, contratistas y trabajadores rurales de la región para medir:

- ¿El problema resuena?
- ¿Hay intención real de usar un servicio así?
- ¿Qué necesitarían para confiar en nosotros?

Si llegaste hasta acá y trabajás en el agro: **tu opinión vale más que cualquier análisis de mercado.**

→ [Anotate en la lista de espera](https://agrolink-oficial.vercel.app)

## 🔭 Visión de producto (cuando haya tracción)

Hoy somos una landing y un formulario. Si la validación confirma las hipótesis, el producto evoluciona hacia:

```
FASE 1 (hoy)          FASE 2                    FASE 3
──────────────        ──────────────────         ─────────────────────
Landing +             Motor de matching          Plataforma rural
lista de espera  →    + perfiles de        →     + integración RENATRE
+ feedback            trabajadores               + historial de campañas
                      + scoring de refs          + capacitaciones online
                      + notif. SMS/WA            + datos para el agro
```

**Stack potencial para escala:**
- 📱 Mobile-first (el campo se maneja por celular)
- 🔔 Notificaciones via WhatsApp Business API
- 🗄️ Base de datos de trabajadores con historial verificado
- 🤝 Integración con RENATRE y Libreta de Trabajo Rural digital
- 📊 Dashboard para productores con disponibilidad en tiempo real
- 🌐 APIs propias con backend robusto (hoy no tenemos nada que escalar todavía — y está bien)
